Privacy policy: Tuki's paper plane.

This privacy policy governs your use of the software application Tuki's paper plane for Android mobile devices that was created by Rocayo.


What information we collect?

Tuki's paper plane does not collect personal information like name, surname, age, date of birth, etc.
We use Google supplied APP performance tracking information to improve Application and "Google Play Games" for leaderboards.

Web requests

Tuki's paper plane performs a web request to grab a "motd" (message of the day) for the title. it also checks which event is currently active.

Spanish motd: https://matyrela.github.io/TukiServerES.html
English motd: https://matyrela.github.io/TukiServerEN.html

Google Analytics

This Application does not use Google Analytics.

This Application does not have any type of Advertisement.
